Node.js: Baixe a versão LTS (Long Term Support) no site oficial node.js.
Git: Instale o Git para Windows git-scm.com.
VS Code: Para editar o código.
npx expo install react-native-calendars

EM SEGUIDA RODAR NO TERMINAL DO VSCODE:

DENTRO DA PASTA DO PROJETO
-- npx expo start // PARA RODAR O EXPO
-- npx expo install @react-navigation/native @react-navigation/stack react-native-screens react-native-safe-area-context react-native-gesture-handler react-native-calendars


-------> CASO O VS CODE N PEÇA O LOGIN
git config --global credential.helper manager

-------> COMO REMOVER O LOGIN DO GIT

O Windows salva sua senha do GitHub em um cofre interno. Para apagar:

No menu Iniciar, pesquise por Gerenciador de Credenciais.

Clique em Credenciais do Windows.

Procure na lista por qualquer item que comece com git:https://github.com.

Clique na seta ao lado e depois em Remover.
2. Deslogar do VS Code
Mesmo limpando o Windows, o VS Code pode manter uma sessão ativa no editor:

No canto inferior esquerdo do VS Code, clique no ícone de Contas (o desenho de uma pessoa).

Se o seu nome de usuário do GitHub aparecer ali, clique em Sair (Sign Out).

3. Limpar as Configurações Globais do Git
O Git costuma salvar seu nome e e-mail para os commits. Para limpar isso no terminal do computador alheio, digite:

Bash
git config --global --unset-all user.name
git config --global --unset-all user.email
